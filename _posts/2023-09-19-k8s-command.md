### get

```bash
# api-resource의 정보를 조회가 가능
# Usage: kubectl get [api-resource] [option]

# node 조회
kubectl get no

# node 상세 조회
kubectl get no -o wide

# pod 조회 (default namespace)
kubectl get po

# pod 상세 조회 (default namespace)
kubectl get po -o wide

# pod 모든 네임스페이스 조회 (--app-namespace or -A)
kubectl get po -A

# pod를 watch 모드로 조회
kubectl get po -w
```
