### api-resources

```bash
# api-resource 목록
kubectl api-resources
```

### get

```bash
# api-resource의 정보를 조회가 가능
# Usage: kubectl get [api-resource] [option]

# node 조회
kubectl get node

# node 상세 조회
kubectl get node -o wide

# pod 조회 (default namespace)
kubectl get pod

# pod 상세 조회 (default namespace)
kubectl get pod -o wide

# pod 모든 네임스페이스 조회 (--app-namespace or -A)
kubectl get pod -A

# pod를 watch 모드로 조회
kubectl get pod -w

# 모든 api-resource 조회
kubectl get all -A
```

### describe

```bash
# api-resource의 상세정보를 조회가 가능
# Usage: kubectl describe [api-resource] [option]

# <node-name>의 상세정보 조회
kubectl describe node <node-name>

# <pod-name>의 상세정보 조회
kubectl describe pod <pod-name>

# <deploy-name>의 상세정보 조회
kubectl describe deploy <deploy-name>

# <pod-name>의 상세정보 조회
kubectl describe pod <pod-name>
```
